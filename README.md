# .NET Repository Template

## How to use

1. Tap the [Use this template](https://github.com/new?template_name=kotlin-js-template&template_owner=seclerp) button
1. Complete repository creation using the template
1. Wait a couple of seconds until the Template Cleanup workflow does its job
1. Clone a re-initialized repository
